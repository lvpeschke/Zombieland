* COMPILATION
Type these instructions in your terminal : 
	# cd <this_directory_path>
	# make


* EXECUTION
Type this instruction in your terminal :
	# make play ARGUMENTS

Allowed arguments are :
--m <file>.ozp	% map file
--z number	% initial number of zombies
--i number	% number items the Brave needs to collect
--b number	% initial number of bullets the Brave has
--help		% display help
